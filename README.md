# ToDo List App
Todo list app made as Minor Project in Coding Ninjas using NodeJS and MongoDB as database.

## Instructions

First clone this repository.
```
$ git clone https://github.com/agentgrey/TodoList.git
```
Install dependencies.
```
$ npm install express
```
Run it
```
$ npm start 
```
